# 🌈 Rainbow Matrix Rain

Uma recriação moderna do clássico efeito **Matrix Rain**, desenvolvida com **HTML5 Canvas, CSS3 e JavaScript**.

Diferente da versão tradicional, onde os caracteres possuem apenas a cor verde, este projeto utiliza o espaço de cores **HSL**, permitindo uma transição contínua entre diversas cores, criando uma animação vibrante e dinâmica.

## Demonstração

O projeto exibe:

* 🌧️ Chuva infinita de caracteres
* 🌈 Transição suave entre múltiplas cores
* ✨ Efeito de rastro (trail)
* ⚡ Animação leve e fluida
* 📱 Layout responsivo para qualquer resolução
* 🎨 Fundo preto inspirado no universo Matrix

##  Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* HTML5 Canvas API

##  Como executar

1. Clone este repositório:

```bash
git clone https://github.com/riqueikki/Rainbow-Matrix-Rain.git
```

2. Entre na pasta do projeto:

```bash
cd Rainbow-Matrix-Rain
```

3. Abra o arquivo `index.html` em qualquer navegador moderno.

Não é necessário instalar dependências ou utilizar servidores.

## 🎨 Personalização

É possível modificar facilmente:

* Velocidade da chuva
* Tamanho da fonte
* Conjunto de caracteres
* Intensidade do rastro
* Saturação das cores
* Velocidade da transição de cores
